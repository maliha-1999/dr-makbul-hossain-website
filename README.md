# 🩺 Dr. Md. Makbul Hossain – Professional Medical Portfolio Website

A responsive and user-friendly personal website designed to showcase the professional profile, services, schedule, and contact information of **Dr. Md. Makbul Hossain**, a specialist in Pediatric Cardiology & Neonatology.

---

## 🌐 Live Preview

👉 *https://anjum-sadia.github.io/Dr-Makbul-s-Portfolio/*

---

## 📌 Project Overview

This project is a multi-page medical portfolio website built using core web technologies. It is designed to provide patients and visitors with clear, accessible, and well-organized information about the doctor’s expertise, services, and availability.

The focus of this project is on **clean UI design, structured layout, and real-world usability**.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (for dynamic content loading)

---

## ✨ Key Features

* Responsive design (mobile, tablet, desktop)
* Multi-page structure (Home, About, Services, Schedule, Gallery, Contact)
* Clean and intuitive navigation system
* Contact form UI for patient inquiries
* Google Maps integration for location
* Reusable footer using JavaScript (`fetch()`)
* Organized layout with consistent styling

---

## 🧠 Development Process

While building this project, I focused on writing clean and maintainable code while simulating a real-world client project.

Key areas of focus:

* Structuring a multi-page website properly
* Maintaining design consistency across pages
* Creating reusable components (e.g., footer)
* Combining Bootstrap with custom CSS for better control
* Organizing assets and files efficiently

This project strengthened my understanding of **frontend development, responsive design, and project structuring**.

---

## ⚙️ Functionality

* Navigation bar provides access to all sections
* “Appointment Now” button redirects users to the contact page
* Contact form collects user input (frontend only)
* Embedded Google Map displays the chamber location
* Footer is dynamically loaded using JavaScript for reuse across pages
